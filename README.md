## в¬Ў What is ZeroHour?

**Zero Hour** вҖ” *the decisive moment where preparation meets performance.*

ZeroHour is a personal, all-in-one command centre for defence exam aspirants. It replaces scattered notes, spreadsheets, and revision apps with a single intelligent platform that tracks everything вҖ” daily logs, mock scores, vocabulary, spaced revision вҖ” and surfaces exactly what needs your attention today.

> Built to be used every single day. Not once a week, not before exam season вҖ” **every day**.

## рҹҡҖ Getting Started

### Prerequisites
- Node.js вүҘ 18
- A Firebase project (Firestore enabled)

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/purvagiri/zerohour.git
cd zerohour

# 2. Install dependencies
npm install

# 3. Set up environment variables
cp .env.example .env
# Open .env and fill in your Firebase config

# 4. Start the development server
npm run dev

# 5. Open http://localhost:5173
```

### Build for Production

```bash
npm run build
# Output in /dist вҖ” ready to deploy
```

> вҡ пёҸ **Never commit `.env` to Git.** It is listed in `.gitignore`.
